### 🎯 典型问答示例

**Q: 如何获取 API Token？**
A: 有两种方式：
   1. 平台认证 Token - 在控制台直接获取
   2. Key Secret 认证 - 使用 API 动态获取
   详见：`ku9i5oea97ynfzt4.md` 和 `tl0hylbi1hm61mu3.md`

**Q: 支持哪些 OCR 能力？**
A: 平台支持多种 OCR 能力：
   - 文本检测/识别 (PPOCR v5)
   - 表格识别 (有线/无线)
   - 公式识别
   - 印章去除
   - 水印去除
   详见：`knowledge/` 下 OCR 相关文档

**Q: 如何使用知识库功能？**
A: 知识库使用步骤：
   1. 创建知识库并上传文档
   2. 配置文本向量化 (embedding)
   3. 使用 RAG 检索 API
   4. 集成到应用中
   详见：`cw6wurwnygkv1nne.md`、`mwhq7ogvirgkxp0l.md`、`caz83r4dkqw6vf5n.md`

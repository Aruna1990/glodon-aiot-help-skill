### 功能二：自动学习

#### 学习脚本 `learn_docs.py`

**核心能力**:
1. 从语雀 API 获取文档内容
2. 解析 Markdown 中的链接
3. 提取语雀知识库内的关联链接
4. 递归下钻学习（可配置深度）
5. 保存到本地 knowledge/ 目录
6. 学习状态持久化

**使用方式**:
```bash
# 学习核心文档
python3 learn_docs.py

# 学习指定文档
python3 learn_docs.py --slugs <slug1> <slug2>

# 学习全部文档
python3 learn_docs.py --all

# 指定下钻深度
python3 learn_docs.py --all --depth 3

# 查看学习状态
python3 learn_docs.py --state
```

**学习流程**:
```
初始文档 → 获取内容 → 提取链接 → 加入队列 → 递归学习 → 保存
   ↓                                        ↑
   └────────────────────────────────────────┘
```

#### 自动触发机制

**触发条件**（在 `bot.py` 中检测）:
```python
# 检测知识盲区
if "不知道" in query or "没有" in query:
    return True, ["lpetkzefs5er4q5x"]  # 常见问题

if "最新" in query or "更新" in query:
    return True, ["bwec6xi60uy9xkgo"]  # 更新记录
```

**用户提示**:
```
抱歉，我暂时没有找到相关信息。

📚 建议操作：
1. 我可以帮您从语雀知识库学习相关文档
2. 查看 [平台使用常见问题汇总](...)
3. 联系技术支持

💡 提示：说"去学习"可以让我更新知识库
```

## 🔧 配置说明

### 环境变量
```bash
export YUQUE_TOKEN='your_yuque_token'
```

### config.yaml 新增配置
```yaml
learning:
  enabled: true           # 是否启用自动学习
  auto_trigger: true      # 检测知识盲区时自动触发
  max_depth: 2            # 最大下钻深度
  rate_limit: 0.5         # 请求间隔（秒）
  state_file: .learn_state.json
  core_docs:              # 核心文档列表
```
